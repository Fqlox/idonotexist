This website tries to show you your doppelganger whose face was made by an artificial intelligence. For a given face, based on your landmarks, the algorithm tries to get the most similar faces, from a dataset of faces renders of thispersondoesnotexist. These images are the result of an Nvidia research project: a StyleGan2 algorithm that generates fake image that was trained of on flickr faces dataset (FFHQ). From its given dataset the algorithm creates new faces that do not exist on its training dataset.
It’s an online version of a student art project made at la HEAR at Strasburg, France. Note that all what the algorithm outputs is uncontrolled and not meant to be taken seriously. No image nor face data is stored in this application.
There is a blog post that explains further how the algorithm works The frontend was built with native js in vitejs. The backend is made in python with flask, face recognition, numpy and pillows.
